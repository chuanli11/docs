    Description¶

    Show list of nodes (machines) and their properties

    Synopsis

    runai top node
    [--details | -d]

    [--help | -h]

    Options¶

    --details | -d

    Show additional details for each node

    Global Flags¶

    --loglevel (string)

    Set the logging level. One of: debug|info|warn|error (default "info")

    --help | -h

    Show help text

    Output¶

    Shows a list of nodes and their properties
